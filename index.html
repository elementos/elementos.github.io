En esta página usted dispone de dos herramientas para química.

Buscador de elementos

Utilice esta herramienta para obtener distintos datos sobre los elementos.

Tabla periódica

Utilice esta herramienta para ver una tabla periódica y recorrerla utilizando el teclado.

Si utiliza NVDA, para desplazarse por los elementos de la tabla mantenga apretadas las teclas CAPSLOCK, SHIFT y alguna de las flechas de dirección

Para ver datos de un elemento particular, presione enter sobre el mismo

Tabla periódica sin espacios

Utilice esta herramienta para ver una tabla periódica sin los espacios en blanco mostrados en la tabla accesible anterior.
